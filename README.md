# Realty Checker Project

Сервис для автоматической проверки объявлений о продаже/аренде квартир.

## Описание проекта
Проект собирает данные с площадок объявлений о недвижимости и предоставляет инструменты для анализа справедливости цены.

## Структура проекта

realty-checker-project/
├── src/ # Исходный код
│ ├── scrapers/ # Модули для сбора данных
│ ├── utils/ # Вспомогательные функции
│ └── config.py # Конфигурация
├── data/ # Данные (под управлением DVC)
│ ├── raw/ # Сырые данные
│ ├── processed/ # Обработанные данные
│ └── external/ # Внешние данные
├── scripts/ # Исполняемые скрипты
├── notebooks/ # Jupyter notebooks для анализа
├── tests/ # Тесты
├── .gitignore # Игнорируемые файлы для Git
├── .dvc/ # Конфигурация DVC
└── README.md # Документация


## Установка и настройка

### Предварительные требования
- Python 3.9+
- uv (пакетный менеджер)
- Git
- DVC

### Шаги по установке

1. Клонировать репозиторий:
```bash
git clone <url-репозитория>
cd realty-checker-project
```

2. Активировать виртуальное окружение:
```bash
source .venv/bin/activate
```

3. Установить зависимости:
```bash
uv pip install -e .
```
