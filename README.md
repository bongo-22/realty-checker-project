# Realty Checker Project

Сервис для автоматической проверки объявлений о продаже/аренде квартир.

## Описание проекта

Проект собирает данные с площадок объявлений о недвижимости и предоставляет инструменты для анализа справедливости цены. В рамках домашнего задания реализован скрейпер для сбора данных с сайта Move.ru.

## Источник данных

**Move.ru** — портал о недвижимости Москвы и Московской области.

Собираемые данные:
- **Структурированные данные**: цена, площадь, этаж, количество комнат, адрес, метро, район
- **Текстовое описание**: свободный текст объявления
- **Изображения**: фото интерьеров и планировок

## Установка и настройка окружения

### Предварительные требования
- Python 3.9+
- uv (пакетный менеджер)
- Git
- DVC

### Шаги по установке

1. **Клонировать репозиторий:**
```bash
git clone https://github.com/ваш-логин/realty-checker-project.git
cd realty-checker-project
```
2. **Создать и активировать виртуальное окружение:**
```bash
uv venv
source .venv/bin/activate  # Linux/Mac
# .venv\Scripts\activate   # Windows
```

3. **Установить зависимости:**
```bash
uv pip install -e .
```

## Сбор данных

1. **Запуск скрейпера**
```bash
python scripts/collect_data.py
```
Скрипт соберет объявления с Move.ru и сохранит их в data/raw/moveru_data.csv

2. **Скачивание фото**
```bash
python scripts/download_images.py
```
Скрипт скачает все фото из CSV в папку data/raw/images/

## Управление данными с DVC

Проект использует DVC для управления данными. Локальное хранилище настроено на ~/dvc-storage

Основные команды:
```bash
# Добавить данные под управление DVC
dvc add data/raw/moveru_data.csv
dvc add data/raw/images

# Отправить данные в хранилище
dvc push

# Забрать данные из хранилища
dvc pull
```
